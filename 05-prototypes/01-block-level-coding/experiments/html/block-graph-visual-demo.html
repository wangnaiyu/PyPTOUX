这页要回答的 3 个问题

What
Block Graph 到底是什么？

它是从 Tile Graph 分区后物化出来的 leaf 子图，每张图都已接近单个 AI Core 的可调度粒度。

How
怎么从 task 跳回 leaf？

先进 root，再用 `callOpMagic` 定位 `CALL`，最后用 `leafHash` 锁定具体 Block Graph。

Why
为什么 `semantic_label` 很关键？

它不是唯一主键，但能稳定连接业务代码、root `CALL` 和 swimlane 事件的人类语义。
