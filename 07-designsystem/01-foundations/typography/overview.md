# Typography

PTO 字体系统面向高密度开发者工作台，而不是营销页面。

## Roles

- Display: 页面级标题，紧凑但清晰
- Title: 面板、卡片和区域标题
- Body: 说明、表单、列表正文
- Label: badge、按钮、metadata
- Mono: 代码、ID、数值、trace key

具体 token 见 `../tokens/foundation.css` 和 `../tokens/components.css`。

## Rule

不要用超大 hero 字号包装工具界面。工作台、graph、timeline 和 inspector 中的标题应保持可扫描、低噪音。
