# Component Specs

本目录承接 PTO 稳定组件规范。当前可执行 class 投影在：

- `../ui/style.css`

## Stable Components

- Button: `.btn`、`.btn-solid`、`.btn-ghost`、`.btn-icon`
- Tabs / segmented control: `.tab-control`、`.segmented-control`
- Panel shell: `.panel-shell`
- Card primitives: `.card-demo`
- Chips / tags: `.stat-chip`、`.nav-stage-chip`、`.detail-type-badge`
- Inputs and local controls

## Rule

不要在 demo 中私造 `.cta`、`.my-button`、`.custom-card` 等视觉组件。找不到等价组件时先走 preview gate。
