# Tools

这里存放服务 PyPTOUX 工作流的内部工具。

当前定位：

- `prototype-kit/previewer/`：解决 JSX demo 预览问题
- `prototype-kit/converters/`：做 HTML / JSX 的轻量转换与包装
- `prototype-kit/launchers/`：快速启动 demo 或课题级预览环境
- `prototype-kit/shared/`：预览器与转换器共用的壳层或运行时辅助
- `scripts/`：轻量脚本
- `templates/`：可复用模板

原则：

- 先轻量起步
- 优先服务设计与原型验证
- 成熟后再升级为完整子项目
