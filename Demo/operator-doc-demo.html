conv2d — 2D 卷积算子
最后同步：2 天前 · 检测到 3 处代码变更
功能描述
对输入张量执行标准二维卷积运算，支持分组卷积与空洞卷积。适用于图像特征提取、深度可分离卷积等场景。
参数说明
kernel_size int Union[int, Tuple[int, int]]
第 1 处 / 共 3 处 — 参数类型扩展
stride Union[int, Tuple] 卷积步长，默认为 1
padding int — 填充大小 Union[int, str] — 填充大小，str 时支持 'same' / 'valid'
第 2 处 / 共 3 处 — padding 新增 str 支持
dilation int 空洞卷积扩张率，默认 1
groups int 分组数，groups=in_channels 时为深度可分离卷积
bias Tensor? 可选偏置项
output_size Tuple? 指定输出空间尺寸，用于转置卷积场景 ⚠ AI 不确定
第 3 处 / 共 3 处 — 新增参数 output_size
返回值
Tensor — 卷积输出，shape 为 (N, C_out, H_out, W_out)
使用示例
import torch
x = torch.randn(1, 3, 224, 224)
w = torch.randn(64, 3, 3, 3)
out = conv2d(x, w, kernel_size=3, padding='same')
