# PR Template | PR 模板

## 标题 | Title

标题用简短的 repo-scoped summary，例如：

- `Reorganize PyPTOUX repository structure`
- `Add JSX previewer scaffold`

## 正文 | Body

正文模板里的 section headers 默认保持英文，便于直接用于 GitHub PR。

```md
## Summary

<一段摘要 / one paragraph summary>

## What changed

- <改了什么 / change>
- <改了什么 / change>

## Why

- <原因 / reason>

## Validation

- <验证项 / check>
```
