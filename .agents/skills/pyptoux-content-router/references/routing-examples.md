# Routing Examples

## Typical Placements

- PyPTO 编译阶段总结
  -> `knowledge/shared/compile-pipeline/overview.md`
- 某个专题的 UX 深度分析
  -> `uxdesign/<topic>/ux-analysis.md`
- Claude 生成的一个 HTML 交互稿
  -> `prototypes/<topic>/experiments/html/<demo-name>.html`
- Claude 生成的 JSX 组件草稿
  -> `prototypes/<topic>/experiments/jsx/<demo-name>.jsx`
- 原型对应的提示词
  -> `prototypes/<topic>/prompts/<demo-name>.md`
- 用于预览 JSX 的小工具
  -> `tools/prototype-kit/previewer/`
- 全局按钮和面板视觉基线
  -> `designsystem/components/specs/` 或 `designsystem/components/ui/`

## File Naming Defaults

- 主题总览 -> `overview.md`
- 来源索引 -> `sources.md`
- 术语表 -> `glossary.md`
- 元数据 -> `meta.md`
- 需求文档 -> `prd.md`
- UX 分析 -> `ux-analysis.md`

## Index Maintenance

- 新课题 -> `docs/indexes/topics.md`
- 新 shared 框架 -> `docs/indexes/shared-frameworks.md`
- 新原型线 -> `docs/indexes/prototypes.md`
- 新工具 -> `docs/indexes/tools.md`
- 新设计系统模块 -> `docs/indexes/designsystem.md`
